# Business Sales Performance Analytics

## 📊 Project Overview
This project analyzes business sales data to understand sales trends, profitability, customer segments, and regional performance using Power BI.

## 🛠 Tools Used
- Power BI
- Microsoft Excel
- Superstore Dataset

## 📌 Key Analysis
- Total Sales & Profit overview
- Sales trend by order date
- Category & sub-category performance
- Regional sales distribution using map
- Segment-wise sales analysis

## 📈 Dashboard Preview
Power BI dashboard includes KPIs, bar charts, line charts, pie charts, and geographic maps.

## 📂 Files Included
- Power BI Dashboard (.pbix)
- Sales Analysis Report
- Dataset (CSV)

## ✅ Outcome
The dashboard provides actionable business insights to improve sales strategy and profitability.
